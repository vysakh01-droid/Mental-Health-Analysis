#  Project

This project contains data analysis and model building using Python.
Analysing mental health and influencing of social media balance in students.

## Objectives

* Analyze the impact of daily screen time on stress and happiness

* Study the relationship between sleep quality and stress

## Files
- project.ipynb : Main code notebook
- Project_Report.docx : Detailed project explanation

## Tools Used
- Python
- Jupyter Notebook
- Pandas, NumPy
- Seaborn, Matplotlib etc.
